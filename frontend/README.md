# SocNet Frontend

Фронтенд для социальной сети SocNet. Работает с API из `../db_api`.

## Запуск

1. Убедитесь, что API запущен (`http://localhost:8080` по умолчанию).
2. Установите зависимости и запустите dev-сервер:

```bash
npm install
npm run dev
```

Приложение будет доступно по адресу http://localhost:3000

## Прокси

Vite проксирует запросы с `/api` на `http://localhost:8080`. Если API на другом порту, измените `vite.config.ts` или задайте `VITE_API_URL` в `.env`.

## Функционал

- **Авторизация**: регистрация, вход, выход (JWT)
- **Посты**: лента подписок, создание постов, ответы на посты, лайки, избранное
- **Пользователи**: просмотр профиля, подписки/отписки, подписчики/подписки
- **Репорты**: жалобы на пользователей
- **Чаты**: список чатов, обмен сообщениями
- **Поиск**: поиск пользователей, начало нового чата
- **Админка** (для is_admin): пользователи, баны, бан по ID репорта, аналитика, логи

## Сборка

```bash
npm run build
```

**Примечание**: Если проект находится в папке с символом `#` в пути (например `C:\Projects\C#\`), сборка `npm run build` может завершаться ошибкой. Dev-сервер при этом работает. Для сборки можно скопировать папку `frontend` в путь без `#`.
